Introduction to EduBridge

Introduction to EduBridge is a class that introduces you to the EduBridge platform. It is a 2-hour class that is conducted every Monday and Wednesday at 10:00 AM. The class is conducted by Mr. Sameer. The class is free of cost and is open to all. The class is conducted in English and is suitable for all age groups. The class is conducted online and the link to join the class is sent to the registered email address.
