EduBridge API Endpoints

Endpoint Name Endpoint Description React Connection
/register/ User registration. This endpoint allows users to register and create an account. In this endpoint we have 3 users {Teacher, Parent, Student}. Each one will register normally except the teacher. After registration, the admin needs to approve the teacher account. Connect to it in React using the appropriate route. For example, you can use the following code in React to make a POST request to the registration endpoint: axios.post('http://127.0.0.1:8000/user/register/')
/profile/ User profile. This endpoint allows users to view and update their profile information. Connect to it in React using the appropriate route. For example, you can use the following code in React to make a GET request to the profile endpoint:axios.post('http://127.0.0.1:8000/user/profile/')
/login/ User login. This endpoint allows users to authenticate and log in to their account. Connect to it in React using the appropriate route. For example, you can use the following code in React to make a POST request to the login endpoint: axios.post('http://127.0.0.1:8000/user/login/')
/logout/ User logout. This endpoint allows users to log out of their account. Connect to it in React using the appropriate route. For example, you can use the following code in React to make a POST request to the logout endpoint: axios.post('http://127.0.0.1:8000/user/logout/')
/forgot-password/ Forgot password. This endpoint allows users to reset their password if they have forgotten it. Connect to it in React using the appropriate route. For example, you can use the following code in React to make a POST request to the forgot password endpoint: axios.post('http://127.0.0.1:8000/user/forgot_password/')
/change-password/ Change password. This endpoint allows users to change their current password in the profile page. Connect to it in React using the appropriate route. For example, you can use the following code in React to make a POST request to the change password endpoint: axios.post('http://127.0.0.1:8000/user/change-password/')
